💻 Digitale Flucht

Alarm im Computerraum: Das Schulnetzwerk wird von Hackern bedroht. Nur wenn alle Sicherheitsrätsel richtig gelöst werden, kann das Netzwerk rechtzeitig gesichert werden.

Deine Mission: Löse 8 Fragenblöcke zu Themen der digitalen Bildung. Aus den richtigen Antworten entsteht pro Block ein Zahlencode mit 3 oder 4 Ziffern. Erst wenn du den passenden Code eingibst, wird der nächste Block freigeschaltet.

So funktioniert das Spiel

  • Beantworte die 3 Fragen eines Blocks.
  • Setze aus den richtigen Antworten den Code zusammen.
  • Gib den Code unten in das Zifferenfeld ein.
  • Nur mit dem richtigen Code geht es weiter.

Bereit, das Netzwerk zu retten?
